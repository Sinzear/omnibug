Omnibug for Chrome

I have created a Chrome version of the Omnibug extension. It functions much like the Firefox version, with the following caveats:

  • By default, shows parameter descriptions (parameter names are shown on hover)
  • All preferences available via the Options page
  • No logging to the filesystem
  • No "watches" sidebar
  • No context menu in panel

It also includes lots more metrics providers, including:

  • Urchin/Google Analytics
  • Google Universal Analytics
  • Adobe/Omniture SiteCatalyst
  • Moniforce (deprecated)
  • WebTrends
  • IBM CoreMetrics
  • AT Internet
  • KISSmetrics
  • Facebook "Like"
  • Torbit
  • Quantcast
  • Marketo
  • Marketo CrowdFactory
  • New Relic
  • Krux
  • Optimizely
  • sophus3

Beta

I am holding a small open beta period for users of the existing Firefox version to try out the Chrome version, and give feedback.

During the beta, please direct all questions, comments, problems and other communications to my email.

  1. Download (right-click -> Save Link As...) the extension: omnibug-0.5.501.crx, released Wed Oct 2 03:56:41 CDT 2013
  2. In Chrome, navigate to Tools → Extensions
  3. Locate the .crx file downloaded in step 1 and drag it into the extensions menu.
  4. Click "Add" at the prompt
  5. Enjoy!

If you choose to download the beta version, please be aware that it is beta quality, and not a finished piece of software. I welcome any feedback you may have!

Note: once the beta period is completed, the extension will be available at the Chrome Web Store.

Use It

  1. Bring up Chrome's DevTools by pressing Ctrl-Shift-I (Mac: Cmd-Shift-I).
  2. In the DevTools panel, click the Omnibug tab.
  3. Load (or reload) a page with analytics tags to decode and display them.

Compatibility

  • Chrome: tested with 26.0.1410.65, but should be backwards compatible several versions
  • OS: tested on OS X 10.7.4 and Windows 7 (should work on any platform that supports Chrome)
